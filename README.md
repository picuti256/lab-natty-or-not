# Click, uma lição de vida

## 📒 Descrição
Um video em formato de youtube, falando um pouco sobre o filme Click e a lição de vida que ele passa.

## 🤖 Tecnologias Utilizadas
Versão gratuita do D-ID e GPT-4.0

## 🧐 Processo de Criação
Foi utilizado o prompt no GPT, pedindo para o mesmo agir como um critico de filmes, falando o impacto que o filme Click tem, junto com sua lição de vida. Após isso, foi utilizado a versão gratuita do D-ID, que infelizmente possui avatares gratuitos bem travados para teste.
